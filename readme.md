## usage

with `start` function in my [`.functions`](https://github.com/armno/dotfiles/blob/master/.functions#L13)

```sh
$ start <DEST>
```

or without

```sh
$ git clone https://github.com/armno/101.git <DEST> && cd <DEST> && rm -rf .git
```
